# Attackability Report

This report evaluates whether generated passwords that score well on zxcvbn are still predictable under a targeted attack model.

## Inputs

- Generated passwords evaluated: 10094
- Targeted candidates generated from common passwords and project rules: 80640000
- Targeted rules: common password + season/admin/test/user phrase + year + symbol, in prefix and suffix forms

## zxcvbn Scores

- Score 4: 10094

## Targeted Attack Matches

- contains_common_password: 9102
- not_matched: 992

Overall targeted-match rate: 90.17%

## Interpretation

The adversarial property is present when a password receives a high zxcvbn score but is still generated by a compact, attacker-controlled rule set. Those matches are much cheaper to guess than their meter score suggests because the attacker does not need to search the full character space.
